# B1点（Buy 1）选股策略

> 创建时间：2026-04-02
> 用途：每日5点自动筛选符合条件的股票

## 筛选条件

| 序号 | 条件 | 说明 |
|------|------|------|
| 1 | **KDJ J值 < 10** | 超卖信号，J值越低越超卖 |
| 2 | **收盘价 < short_line** | 股价在短期趋势线下方（短期趋势偏弱） |
| 3 | **收盘价 > big_line** | 股价在多空线上方（中长期趋势尚可） |
| 4 | **三个月内无顶部放巨量阴线** | 避免主力出货形态 |
| 5 | **目前为缩量下跌** | 下跌动能衰减，可能见底 |

## 技术指标定义

- **short_line** = EMA(EMA(Close, 10), 10) — 知行短期趋势线
- **big_line** = (MA14 + MA28 + MA57 + MA114) / 4 — 知行多空线
- **kdj_j** = 3×K - 2×D（KDJ的J值）
  - RSV = (Close - Low9) / (High9 - Low9) × 100
  - K = SMA(RSV, 3)
  - D = SMA(K, 3)
  - J = 3K - 2D

## 条件详细说明

### 条件4：三个月内无顶部放巨量阴线
- **顶部**：近3个月最高价附近（±5%）
- **巨量**：成交量超过近20日均量的2倍
- **阴线**：收盘价 < 开盘价
- 满足以上三点的交易日视为"顶部放巨量阴线"，需排除

### 条件5：目前为缩量下跌
- **缩量**：近5日均量 < 近20日均量 × 0.8
- **下跌**：近5日股价呈下跌趋势（线性回归斜率为负）
